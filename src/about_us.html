The Bedu Project

This project is a web recipe search engine, consuming the TheMealDB API. It is possible to generate searches through the interface and display the results dynamically. When accessing a recipe, more details about it, such as ingredients and instructions, are displayed. Additionally, it is possible to search for recipes in a random way.
